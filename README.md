# Data Science Jobs Analysis Dashboard

## Project Overview
This project analyzes global Data Science job trends using Excel. The dashboard provides insights into salary distribution, job roles, employment types, and country-wise salary trends.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Visualization
- Dashboard Design

## Key Insights
- Data Scientist roles have the highest salary among the top job titles.
- Most jobs are Full-Time positions.
- The United States offers the highest average salary for data science roles.
- Salary trends have increased from 2020 to 2022.
- Mid-level professionals earn significantly higher salaries than entry-level roles.

## Dashboard Features
- Interactive filters for **Work Year**
- Country-wise salary comparison
- Job title salary analysis
- Employment type distribution
- Salary trend visualization
